利多出盡

「利多出盡」是指一項好消息（例如公司發布亮眼財報或利好消息）公佈後，股價反而下跌的現象。這通常是因為市場預期已經提前反應了這項利多，當消息正式公佈時，買盤動能減弱，甚至有部分投資人選擇獲利了結，導致股價下跌。
