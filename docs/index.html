利多出盡

「利多出盡」是指當一項對公司或市場有利的消息（利多）被公開後，股價反而不漲反跌的現象。這通常是因為市場預期已將該利多反應在股價中，甚至過度反應，當消息真正公布時，買盤力道減弱，反而引發獲利了結賣壓。
