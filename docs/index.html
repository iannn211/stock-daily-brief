利多出盡 (Buy the rumor, sell the news)

「利多出盡」是指一項好消息或預期中的利多事件發生後，股價反而下跌的現象。這通常是因為市場在消息公布前就已經提前反應了這項利多，當消息真正公布時，買盤動能減弱，甚至有投資人選擇獲利了結，導致股價回檔。台積電昨日 (4/17) 法說會後股價不漲反跌，就是一個典型的利多出盡例子。
