📚 新手學習點

利多出盡

「利多出盡」是指一項被市場普遍看好的利好消息（例如公司財報優於預期、產業前景光明）公布後，股價反而下跌的現象。這通常是因為股價在消息公布前就已經充分反映了這些利好，當消息真正公布時，買盤力道減弱，甚至有獲利了結的賣壓出現，導致股價回落。這提醒我們，好消息不一定會讓股價繼續漲，有時反而會成為短期高點。
